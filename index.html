My Projects

Hot This Weekend

"Hot this Weekend" is an app aimed at people of all ages who enjoy concerts and events. The user will enter in a zip code and this app will return concerts, events and 5 days worth of weather to allow people to plan their concerts and events accordingly based on the weather.

How it works:

Hot This Weekend retrieves and displays information from API calls made through JQuery's ajax method. The JSON formatted data is displayed to the user using bootstrap, which also utilizes libraries such as animate.css.

Optimum Auto Styling

"O.A.S" is an e-commerce website focused on offering customers our car products and auto styling services.

How it works:

Customers can submit an appointment with the services provided by O.A.S. The appointment gets stored in our MongoDB and when the appointment is booked, the customer is immediately emailed the details. It retrieves information from MongoDB using mongoose as an ODM to display products for customers to add to the cart and purchase. It is launched on Node.js and Express server which is hosted on Heroku with MongoLab services.

H.A.S. Blood Clinic

H.A.S. Blood Clinic is a scheduling application which allows a potential donor to sign up to donate blood and make appointments on when they would like to do so. Our application is also user-friendly as it has more information on what to do if one would like to volunteer for H.A.S. Blood Clinic, host a blood drive, or even donate money!

How it works:

H.A.S. Blood Clinic uses mySQL to store the information of a donor to make it easier for our clinic to handle the donors who wish to donate blood and give back to the community.
