My Projects

Hot This Weekend

"Hot this Weekend" is an app aimed at people of all ages who enjoy concerts and events. The user will enter in a zip code and this app will return concerts, events and 5 days worth of weather. The purpose of this is to allow people to plan their weekend based around an event and the kind of weather they can expect.

How it works

Hot This Weekend retrieves and displays information from API calls made through JQuery's ajax method. The JSON formatted data is displayed to the user using bootstrap, which also utilizes libraries such as animate.css.




Optimum Auto Styling

"O.A.S" is an E-commerce website focused on reaching out and offering customers our products and services. As we were building this website, an admin portal was created for the staff to keep track of invoices, sales, and number of services completed.

How it works

Customers can submit an appointment form with a selection of one or many services offered by O.A.S. The appointment gets stored in our MongoDB and with the implementation of email.js, the company shoots them an email about their current appointment. It retrieves information from mongoDB using mongoose as an ODM to display products for customers to add to the cart and of course to buy! It is launched on a Node.js Express server that is hosted on Heroku along with MongoLab service.


H.A.S. Blood Clinic

H.A.S. Blood Clinic is a scheduling application which allows a potential donor to sign up to donate blood and make appointments on when they would like to do so. Our application is also user-friendly as it has more information on what to do if one would like to volunteer for H.A.S. Blood Clinic, host a blood drive, or even donate money!.

How it works

H.A.S. Blood Clinic uses mySQL to store the information of a donor to make it easier for our clinic to handle the donors who wish to donate blood and give back to the community.
